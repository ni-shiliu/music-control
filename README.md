# neil-control — Claude Code Skills

A collection of Claude Code skills for macOS productivity.

**Skills included:**

- `music-control` — Proactively controls music playback on macOS (KuGou or Spotify) based on conversation vibe.

More skills coming soon.

## Install

```bash
git clone https://github.com/ni-shiliu/neil-control ~/.claude/skills/neil-control
```

Claude Code will automatically load the skills on next startup.
